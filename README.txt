退職・老後の手続きチェックナビ v1.8

■内容
質問に答えると、退職・老後に必要になりそうな手続きや確認事項を整理する単体HTMLアプリです。
結果は制度上の確定判定ではなく、勤務先・自治体・年金事務所などへ確認するためのチェックリストです。

■v1.8の主な変更点
・アプリ内に著作権・転載に関する注意を追加しました。
・無断転載、複製、再配布、改変しての公開を控えていただく旨を明記しました。
・個人利用のための印刷・保存、家族や相談窓口へ見せるためのチェックリスト利用、リンクでの紹介は問題ないことを明記しました。
・公的制度の名称や事実そのものの利用を制限するものではないことを明記しました。
・バージョン表記を「v1.8 / PCブラウザ推奨」に変更しました。
・保存データ内のバージョン値を「1.8.0」に変更しました。

■継続している主な仕様
・PCブラウザ利用を前提としています。
・スマホは動作する場合がありますが、表示・保存データ読込・印刷レイアウトは端末やブラウザにより異なります。
・氏名、住所、マイナンバー、基礎年金番号、銀行口座番号などは入力しない前提です。
・入力内容やメモはアプリから外部送信しません。ただし、GitHub Pages表示や外部リンク利用時の通信は発生します。
・保存ファイルは暗号化されず、メモ帳などで確認できます。
・保存ファイルはブラウザのダウンロード先に保存されます。
・詳細一覧印刷では完了済み項目を印刷しません。
・サマリー印刷では完了済み項目も表示し、グレー表示＋取り消し線で区別します。
・生まれ年・誕生月が未入力の場合も、年齢に関係する確認項目を安全側に残します。

■著作権・転載について
本アプリの文章、画面構成、デザイン、コードの無断転載、複製、再配布、改変しての公開はお控えください。
個人利用のための印刷・保存、家族や相談窓口へ見せるためのチェックリスト利用、リンクでの紹介は問題ありません。
公的制度の名称や事実そのものの利用を制限するものではありません。

■公開方法
GitHub Pagesで公開する場合は、GitHub公開用ZIPを展開し、リポジトリのルートへ index.html と .nojekyll を配置してください。

■注意
制度や手続きは変更されることがあります。また、自治体・勤務先・加入制度により扱いが異なります。
実際の手続き前には、必ず最新情報を各窓口で確認してください。
