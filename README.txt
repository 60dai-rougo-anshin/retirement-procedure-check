v2.5の変更点:
- 健康保険の高額療養費・限度額適用認定と、所得税の医療費控除を別質問に分離。
- 所得税の還付申告だけの場合に、健康保険側の医療費軽減申請が出ないよう修正。
- 健康保険側の質問文から「医療費控除」を削除し、手続きの混同を防止。

注意:
- 本アプリは申請・届出・請求・加入・切替などの漏れを防ぐためのチェックリストです。
- 制度上の該当可否や金額を確定するものではありません。
- 最終判断は、勤務先、市区町村、年金事務所、税務署、加入中の健康保険者などの公的窓口で確認してください。
