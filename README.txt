60代からの手続き漏れチェックナビ v2.6

このZIPはGitHub Pages公開用です。

アップロードするファイル:
- index.html
- .nojekyll
- README.txt

使い方:
1. GitHubリポジトリの直下に、この3ファイルをアップロードします。
2. Settings > Pages で Source を Deploy from a branch、Branch を main / root にします。
3. 公開URLを開くと index.html が表示されます。

注意:
- README.txt は利用者向け画面には自動表示されません。
- 利用者に読ませたい注意事項はアプリ画面内に記載済みです。
- 入力した情報はブラウザ内で判定します。外部送信はしません。
- 保存JSONは暗号化されません。重要な個人情報は入力しないでください。
